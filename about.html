About This Project

Tic Tac Toe is a clean, minimal, mobile-first web game crafted for speed, accessibility, and joy. It features fire animation, responsive layout, and sound — designed to feel native on iPhones and MacBooks.

Who Made This?

This project was built by edencry, a developer blending creative design and spiritual purpose. It is part of a broader mission to inspire through interactive, aesthetic experiences.
